# Proyecto 01 (Pensamiento computacional)
Descripción del sistema
Este proyecto consiste en el desarrollo de un sistema que permite gestionar contenido de una plataforma de streaming.
El sistema permite registrar información sobre diferentes tipos de contenido, como películas o series, así como controlar datos importantes como duración, clasificación, nivel de producción y estado de publicación.
Además, el sistema valida los datos ingresados por el usuario, asegurando que sean correctos antes de almacenarlos o procesarlos.

Instrucciones para ejecutarlo
Abrir el entorno de desarrollo (por ejemplo, Visual Studio o VS Code).
Cargar el archivo del proyecto.
Compilar el programa para verificar que no existan errores.
Ejecutar el sistema.
Seguir las instrucciones que aparecen en consola:
Ingresar datos solicitados (tipo de contenido, duración, clasificación, etc.)
Elegir opciones del menú según se indique
Asegúrese de ingresar valores válidos (números donde se soliciten números, texto donde se requiera texto), ya que el sistema incluye validaciones.

Explicación del proyecto
El sistema fue desarrollado utilizando estructuras básicas de programación, principalmente en C#.
Se implementaron:
Variables para almacenar información del contenido
Estructuras de control como if, switch, while y for
Validaciones con TryParse para evitar errores de entrada
Menú interactivo para facilitar la navegación del usuario
